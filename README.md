# Easy Anti-Cheat Runtime Log Analyser

This project is a lightweight Python prototype developed as part of cybersecurity research into kernel-level anti-cheat systems.

The prototype analyses ProcMon CSV logs collected during Fortnite and Easy Anti-Cheat runtime and generates simple behavioural summaries related to process activity and system interaction.

## Features

- ProcMon CSV log analysis
- Process activity summaries
- Operation counting
- Automatic CSV export
- Simple behavioural visualisation

## Technologies Used

- Python
- Pandas
- Matplotlib

## Usage

1. Place the ProcMon CSV file in the same directory as the script.
2. Run:

```bash
python eac_log_analyser.py
